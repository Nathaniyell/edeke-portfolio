SmartFunds - Fintech App Design

Project Overview

  • Project Name:   SmartFunds
  • Role:   UI/UX Designer/Researcher
  • Duration:  8 weeks
  • Tools Used:  Figma
  • Project Type:  Self-initiated, skill enhancement project

Project Goal

The goal of this project was to design an intuitive and inclusive fintech app interface that seamlessly enables users to:

  • Send and receive money
  • Access financial education
  • Explore investment and loan options
  • Manage bill payments
  • Set savings goals
  • Interact with AI features
  • Navigate the app in multiple languages

    • All of these features were built to offer a user-friendly and engaging financial experience.

My goals for my own personal development:

  • Learn how to conduct and analyze user research, create flow charts and wireframes, design and prototype through Figma
  • Complete my first redesign project from start to finish while sticking to design principles

The roles I assumed during the process of building this redesign:

  • UI/UX Designer
  • User Researcher
  • Product Designer
  • Data Analyst
